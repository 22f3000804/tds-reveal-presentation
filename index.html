import pandas as pd
import numpy as np

def calculate_roi(initial_investment, final_value):
    """Calculate Return on Investment"""
    roi = ((final_value - initial_investment) / initial_investment) * 100
    return roi

def compound_annual_growth_rate(start_value, end_value, years):
    """Calculate CAGR for investment analysis"""
    cagr = (pow(end_value / start_value, 1 / years) - 1) * 100
    return cagr

Financial Formulas

Compound Interest:

\[ A = P \left(1 + \frac{r}{n}\right)^{nt} \]

Return on Investment (ROI):

\[ ROI = \frac{(Final Value - Initial Investment)}{Initial Investment} \times 100\% \]

Compound Annual Growth Rate (CAGR):

\[ CAGR = \left(\frac{V_{final}}{V_{initial}}\right)^{\frac{1}{n}} - 1 \]

These are the key financial formulas used in our analysis. A = final amount, P = principal, r = rate, n = compounding frequency, t = time. Explain each formula as it appears with the fragments.
